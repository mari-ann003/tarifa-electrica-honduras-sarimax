# tarifa-electrica-honduras-sarimax
Predicción econométrica de la tarifa eléctrica en Honduras usando modelos SARIMAX

# Disponibilidad de datos

Los conjuntos de datos utilizados en este proyecto no pueden ser redistribuidos
por restricciones institucionales y legales.

## Fuentes de datos

- Pliegos de ajuste tarifario:
  Comisión Reguladora de Energía Eléctrica (CREE), Honduras  
  Período 2021–2026

- Tipo de cambio (HNL/USD):
  Banco Central de Honduras (BCH)  
  Serie mensual de tipo de cambio promedio

## Notas

Para reproducir el análisis, el usuario debe descargar manualmente los datos
desde las fuentes oficiales y ejecutar los scripts de preprocesamiento
incluidos en el repositorio.
